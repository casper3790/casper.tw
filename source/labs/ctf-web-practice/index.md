# CTF Web Practice

- **Status:** Archived
- **Date:** 2026-06-02

## Purpose

Record practice around **CTF**, Web Security, Linux basics, and OSINT.

## Setup

- Browser developer tools
- Linux terminal
- Basic HTTP inspection tools
- Practice challenges

## Steps

1. Read the challenge statement carefully.
2. Inspect the web page, request, and response.
3. Search for weak assumptions.
4. Write down the solution path and mistake points.

## Result

The practice improved my ability to slow down, inspect details, and avoid jumping to conclusions too quickly.

## Notes

The most useful part is not only solving the challenge, but writing a clean explanation after solving it.
