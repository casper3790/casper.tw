# コミュニティ・教育

## Overview

コミュニティ活動は私の学習の一部です。授業、コンペティション、ワークショップが終わった後も、技術学習を続ける場所になります。

## Timeline

- **NIHSCEDIWHC White Hat Community** - Cybersecurity community building と知識共有。
- **Cybersecurity community building** - 学生が security topics を話し、練習できる場づくり。
- **Cybersecurity Teaching Assistant** - 練習セッションと技術質問の支援。
- **Student Representative** - コミュニケーションと学生活動の調整。
- **CTF training** - Web Security、Linux、OSINT、challenge solving の練習支援。
- **Raspberry Pi teaching** - Linux、GPIO、Python、embedded experiments の学習支援。

## Notes

コミュニティ活動は実用的であるべきだと考えています。「Cybersecurity を学ぶ」と言うだけでなく、繰り返し練習できる道筋を作ることが目標です。
