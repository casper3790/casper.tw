# ワークショップ

## Overview

ワークショップは、自分のプロジェクト以外の人たちが **Cybersecurity**、cloud、AI、open source、healthcare、manufacturing systems をどう考えているかを知る機会です。

## Timeline

- **HITCON** - Security community と conference learning。
- **COSCUP** - Open source community と技術共有。
- **AWS Summit Taiwan** - Cloud architecture、infrastructure、developer workflows。
- **Build with AI** - AI application patterns と developer tooling。
- **GDG Taipei** - Google developer community events と hands-on workshop。
- **Gemini CLI + ADK + Cloud Run** - AI agent と deployment workflow。
- **AI × Cybersecurity Workshop** - AI tooling と security practice の接続。
- **Quantum Computing Summer Camp** - Quantum Computing concepts と実験入門。
- **Future Medical Design Hackathon** - Healthcare problem framing と prototype thinking。
- **Smart Manufacturing Training** - 産業向け automation と system thinking。
