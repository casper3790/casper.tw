# 資格

## Overview

このページでは、学習背景の一部として certifications と licenses を整理しています。

## Cards

### Computer Hardware Installation Technician Level C

computer hardware installation、maintenance concepts、実務的な setup の基礎。

### Computer Network Installation Technician Level C

network installation、cabling、configuration concepts、troubleshooting の基礎。

### Certified Lifeguard

safety、responsibility、emergency response に関する training。

### World Taekwondo Black Belt 3rd Dan

discipline、physical practice、persistence を伴う長期的な training record。

## Notes

これらの資格はすべて software に直接関係するものではありませんが、実践的な training、責任感、長期的な努力を示す記録です。
