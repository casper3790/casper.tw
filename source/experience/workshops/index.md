# Workshops

## Overview

Workshops help me see how people outside my own projects think about **Cybersecurity**, cloud, AI, open source, healthcare, and manufacturing systems.

- **HITCON** - Security community learning and conference exposure.
- **COSCUP** - Open source community and technical sharing.
- **AWS Summit Taiwan** - Cloud architecture, infrastructure, and developer workflows.
- **Build with AI** - AI application patterns and developer tooling.
- **GDG Taipei** - Google developer community events and practical workshops.
- **Gemini CLI + ADK + Cloud Run** - AI agent and deployment workflow learning.
- **AI × Cybersecurity Workshop** - Connecting AI tooling with security practice.
- **Quantum Computing Summer Camp** - Introduction to quantum computing concepts and experiments.
- **Future Medical Design Hackathon** - Healthcare problem framing and prototype thinking.
- **Smart Manufacturing Training** - Industry-oriented automation and system thinking.

## Notes

I use workshops as input for future projects. Not every workshop becomes a project immediately, but each one helps me understand another technical field more concretely.
