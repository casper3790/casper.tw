# 關於

## Intro

我是 **Casper**，本名 **洪新富**。

我是在台灣學習工程與資訊安全的學生，目前專注在 **Cybersecurity**、**Web development**、**Robotics**、**Embedded systems** 與 **Medical Cybersecurity**。這個網站不是一份包裝好的公司履歷，比較像是一個還在成長中的工程學生，公開記錄自己如何做專案、參加比賽、debug、失敗，再繼續修正的地方。

## What I Build

我喜歡把系統真的做出來，而不是只停留在概念。現在的學習方向會把 web platform、安全思維、Robotics 實驗、embedded devices 與醫療場域的資安問題串在一起。

我常回到這些題目：

- Cybersecurity 與 CTF 練習
- 使用 FastAPI 與 frontend workflow 做 web systems
- ROS2、Raspberry Pi、ESP32 與 Robotics prototypes
- Medical Cybersecurity 與 data validation 問題
- 幫助團隊整理技術工作的工具

## How I Learn

我主要透過實作學習。比賽、研習、失敗的實驗、debug 過程、專案筆記，都是學習紀錄的一部分。

當系統壞掉時，我會盡量寫下發生什麼、我誤解了什麼、最後改了什麼。這件事對我很重要，因為它能把零散的努力變成之後可以回頭看的紀錄。

## Why This Site Exists

這個網站是我的 **public engineering notebook**。

我用它實踐 **Build in Public**，整理 Projects 和 Labs，也把學習路上的真實樣子留下來。有些頁面比較完整，有些還是筆記，但重點是持續前進，讓學習過程被看見。

## My Quote

> 如果你真的認為這是你的路，那就不要半途而廢。
