# 社群與教學

## Overview

社群工作是我學習的一部分。它讓技術學習不會在課程、比賽或研習結束後就中斷。

## Timeline

- **NIHSCEDIWHC White Hat Community** - Cybersecurity community building 與知識分享。
- **Cybersecurity community building** - 建立讓學生討論 security topics 並一起練習的空間。
- **Cybersecurity Teaching Assistant** - 支援練習課程與技術問題。
- **Student Representative** - 協助溝通與學生相關活動。
- **CTF training** - 支援 Web Security、Linux、OSINT 與解題練習。
- **Raspberry Pi teaching** - 協助學生學習 Linux、GPIO、Python 與 embedded experiments。

## Notes

我希望社群工作保持實際。目標不只是說「學 Cybersecurity」，而是建立可以重複練習的路徑。
