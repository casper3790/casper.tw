# 研習

## Overview

研習讓我看到自己專案之外的人如何思考 **Cybersecurity**、cloud、AI、open source、healthcare 與 manufacturing systems。

## Timeline

- **HITCON** - Security community 與 conference learning。
- **COSCUP** - Open source 社群與技術分享。
- **AWS Summit Taiwan** - Cloud architecture、infrastructure 與 developer workflows。
- **Build with AI** - AI application patterns 與 developer tooling。
- **GDG Taipei** - Google developer community events 與實作 workshop。
- **Gemini CLI + ADK + Cloud Run** - AI agent 與部署流程學習。
- **AI × Cybersecurity Workshop** - 連結 AI tooling 與 security practice。
- **Quantum Computing Summer Camp** - Quantum Computing 概念與實驗入門。
- **Future Medical Design Hackathon** - Healthcare problem framing 與 prototype thinking。
- **Smart Manufacturing Training** - 產業導向的 automation 與 system thinking。

## Notes

我會把研習當成未來專案的輸入。不是每一場研習都會立刻變成作品，但每一場都讓我更具體理解一個技術領域。
